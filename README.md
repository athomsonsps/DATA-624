DATA 624: Predictive Analytics
Repository Overview

This repository contains coursework for DATA 624: Predictive Analytics in the M.S. in Data Science program at CUNY School of Professional Studies.

The course focuses on predictive modeling, forecasting, model evaluation, and advanced regression methods. The work in this repository includes homework assignments, R Markdown files, reports, model outputs, and supporting files used for class submissions.

Notes

This repository is for academic coursework and documentation. The reports are written to be clear, direct, and easy to follow. Each assignment is organized so the reader can understand the problem, the code, the results, and the final interpretation.
