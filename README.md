This assignment is about preparing data before building classification models. The main goal is to inspect two datasets, understand their problems, and decide what preprocessing steps are needed before modeling.

For KJ 3.1, the focus is the Glass dataset. The dataset contains glass samples with chemical measurements such as refractive index, sodium, magnesium, aluminum, silicon, potassium, calcium, barium, and iron. The goal is to study the predictor variables before using them to classify glass type. I checked the class distribution, predictor distributions, boxplots, correlations, skewness, and outliers. These checks helped show which variables were unevenly distributed, which variables had extreme values, and which variables were strongly related to each other. The main preprocessing lesson from this section is that some predictors may need transformation, centering, scaling, or closer review before they are used in a classification model.

For KJ 3.2, the focus is the Soybean dataset. This dataset has many categorical predictors and several disease classes. The goal is to study how the categories are distributed, identify predictors with little useful variation, and understand where missing values occur. I checked class balance, frequency distributions, near-zero variance predictors, missing values by variable, and missing values by class. This helped show that missing data was not spread evenly across the dataset. Some disease classes had much more missing data than others, which means deleting rows could damage the dataset and make class imbalance worse. The main preprocessing lesson from this section is that categorical data needs careful handling, especially when predictors have missing values or very little variation.

Overall, this assignment teaches that model building should not start until the data is understood. Good preprocessing decisions come from checking the structure of the data, finding imbalance, identifying skewness and outliers, reviewing missing values, and deciding whether to transform, scale, remove, or impute variables. The purpose is to make the data cleaner and more reliable before using it for predictive modeling.
